# AI-Ethics-Awareness-Keyword-Analysis-Pivot-Table

This project applies data analysis techniques using Microsoft Excel to support an Honours-level research study focused on AI ethics awareness in work-integrated learning policy and practice. The analysis was conducted to identify patterns in how frequently key governance-related concepts appear within academic literature.

Raw data was manually extracted from peer-reviewed journal articles aligned with the research topic and then cleaned, organised, and structured in Excel to enable systematic analysis.

# Analysis Focus
The project analysed the frequency of governance-related keywords within academic sources to understand how strongly certain themes are emphasised in AI discourse. The key terms analysed were:

1. Ethics
   
2. Practice
   
3. Policy
   
The goal of the analysis was to evaluate whether ethical considerations in AI are sufficiently represented in discussions surrounding work-integrated learning, institutional policy, and professional practice.

# Tools and Techniques Used
• Microsoft Excel

• Data cleaning and structuring of extracted research data

• Keyword frequency analysis

• Pivot table analysis to summarise thematic patterns

• Research data organisation and validation

# Project Output
The project includes:

• A structured dataset derived from peer-reviewed academic sources

• A pivot table used to analyse keyword frequency across the research dataset

• Analytical insights supporting the broader research investigation

# Key Insight
The pivot table analysis showed a higher frequency of the term “ethics” within the analysed dataset, reinforcing the importance of ethical governance frameworks for AI adoption, particularly within academic institutions and professional workplaces.

These findings support the broader research argument that AI ethics awareness should play a central role in educational policy, institutional governance, and work-integrated learning environments.
